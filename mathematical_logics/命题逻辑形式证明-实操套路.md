# 命题逻辑形式证明 · 实操套路

> 格式谁都会写（序号 | 公式 | 依据），真正卡人的是：**面对一堆前提，第一行该写什么？**

---

## 核心心法：先看结论长什么样

形式证明不是"从前提乱推"，而是**从结论倒推，再正着写**。拿到题以后，先别看前提，先盯着结论，问自己一句话：

> 最后一步用什么规则，能得到这个结论？

结论的形状直接决定了证明策略。下面按结论的四种常见形状来讲。

---

## 套路一：结论是蕴含式 $A \to B$

**策略：附加前提法（CP）**

结论长成 $A \to B$，说明你最后一步一定是「撤去附加前提」。那倒数第二步就必须已经得到了 $B$。

**操作流程：**

1. 先把原始前提照抄引入
2. 附加 $A$ 作为新前提
3. 用 $A$ + 原有前提，设法推出 $B$
4. 撤去 $A$，得到 $A \to B$

**例题**：$\{p \to (q \to r)\} \vdash q \to (p \to r)$

| 步骤 | 公式 | 依据 |
| :-: | :--- | :--- |
| ① | $p \to (q \to r)$ | 前提引入 |
| ② | $q$ | 附加前提 |
| ③ | $p$ | 附加前提 |
| ④ | $q \to r$ | ①③ 假言推理 |
| ⑤ | $r$ | ②④ 假言推理 |
| ⑥ | $p \to r$ | 撤去 ③ |
| ⑦ | $q \to (p \to r)$ | 撤去 ② |

**心法**：结论有几层 $\to$，就附加几次前提，从最外层往里面剥。

---

## 套路二：结论是原子命题（如 $q$、$r$）

**策略：回溯——谁手里有这个结论？**

结论不是一个带 $\to$ 的大公式，而是一个单独的 $q$。这时问自己：

> 哪个前提里出现了 $q$？

可能出现的情况：

| 前提中含有 | 对应操作 |
|---|---|
| $p \to q$ | 想办法拿到 $p$，然后假言推理出 $q$ |
| $p \lor q$ | 想办法拿到 $\neg p$，然后析取三段论出 $q$ |
| $q \land r$ | 直接化简式出 $q$ |

**例题**：$\{p \lor q,\ p \to r,\ \neg r\} \vdash q$

| 步骤 | 公式 | 依据 |
| :-: | :--- | :--- |
| ① | $p \lor q$ | 前提引入 |
| ② | $p \to r$ | 前提引入 |
| ③ | $\neg r$ | 前提引入 |
| ④ | $\neg p$ | ②③ 拒取式 |
| ⑤ | $q$ | ①④ 析取三段论 |

**心法**：结论是 $q$，前提里有 $p \lor q$，那我只要把 $p$ 干掉（拿到 $\neg p$）就行了。

---

## 套路三：结论是否定式 $\neg A$

**策略：归谬法**

假设 $A$ 成立，然后推出矛盾。

**操作流程：**

1. 引入所有前提
2. 附加 $A$（假设结论的反面）
3. 往前推，直到推出形如 $B \land \neg B$ 的矛盾
4. 归谬，得到 $\neg A$

**例题**：$\{p \to q,\ \neg q\} \vdash \neg p$

| 步骤 | 公式 | 依据 |
| :-: | :--- | :--- |
| ① | $p \to q$ | 前提引入 |
| ② | $\neg q$ | 前提引入 |
| ③ | $p$ | 附加前提（假设结论不成立） |
| ④ | $q$ | ①③ 假言推理 |
| ⑤ | $q \land \neg q$ | ②④ 合取（矛盾） |
| ⑥ | $\neg p$ | ③⑤ 归谬 |

**心法**：直接推不出 $\neg p$ 的时候，反问"如果 $p$ 真的成立会怎样"，往往立刻撞墙。

---

## 套路四：结论是析取式 $A \lor B$

**策略 A**：如果前提里有 $A \lor B$ 或类似的析取，可能是 构造性二难。

**策略 B**：更常见的——先推出 $A$（或 $B$），然后 附加式 得到 $A \lor B$。

**例题**：$\{p \to r,\ p \lor q,\ q \to s\} \vdash s \lor r$

| 步骤 | 公式 | 依据 |
| :-: | :--- | :--- |
| ① | $p \lor q$ | 前提引入 |
| ② | $p \to r$ | 前提引入 |
| ③ | $q \to s$ | 前提引入 |
| ④ | $r \to (s \lor r)$ | 附加式 |
| ⑤ | $s \to (s \lor r)$ | 附加式 |
| ⑥ | $p \to (s \lor r)$ | ②④ 假言三段论 |
| ⑦ | $q \to (s \lor r)$ | ③⑤ 假言三段论 |
| ⑧ | $s \lor r$ | ①⑥⑦ 构造性二难 |

**心法**：前提里有一个 $p \lor q$，两个蕴含分别指向 $r$ 和 $s$，典型的"分情况讨论"结构，构造性二难直接拿下。

---

## 做题时的决策流程（重要！）

拿到一道证明题，按下面顺序过一遍：

```
1. 结论是 A → B 吗？
   → 是：附加前提法，先加 A
   → 否：往下看

2. 结论是 ¬A 吗？直接推推不动吗？
   → 是：归谬法，先加 A
   → 否：往下看

3. 前提里有析取 A ∨ B 吗？
   → 有：要么析取三段论（配合 ¬A 或 ¬B），要么构造性二难（配合两个蕴含）
   → 没有：往下看

4. 盯着结论里的原子命题（如 q），找前提里谁包含它
   → 它在某个蕴含的后件？→ 想办法拿到前件
   → 它在某个析取里？→ 想办法干掉另一个
   → 它在某个合取里？→ 直接化简
```

## 最常出现的几个"固定搭配"

| 你手里有 | 你想要 | 做法 |
|---|---|---|
| $A \to B,\ A$ | $B$ | 假言推理 |
| $A \to B,\ \neg B$ | $\neg A$ | 拒取式 |
| $A \lor B,\ \neg A$ | $B$ | 析取三段论 |
| $A \land B$ | $A$（或 $B$） | 化简式 |
| $A \to B,\ B \to C$ | $A \to C$ | 假言三段论 |
| $A$ | $A \lor B$ | 附加式 |
| $A \leftrightarrow B$ | $A \to B$ | 双条件分解 |

## 最后的建议

形式证明说白了就是**规则匹配游戏**。你的任务不是"发明"推理路径，而是：

1. 把前提摊开——手里有哪些牌
2. 盯着结论——我要去哪
3. 匹配规则——哪条规则能把我手里的牌变成我要的结论

每写一行之前，只问一句话：**"这一行的依据是什么？为什么现在能做这一步？"**

答不上来就别写，先回头看前提和规则表。熟练了以后，很多题三五行就结束了。
