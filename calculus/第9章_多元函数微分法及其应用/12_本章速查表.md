# 第9章 多元函数微分法及其应用

## 本章速查表

| 主题 | 核心公式 | 什么时候用 |
| --- | --- | --- |
| 偏导数 | $f_x,\ f_y$ | 只让一个变量变化 |
| 全微分 | $dz=f_xdx+f_ydy$ | 多个变量同时微小变化 |
| 复合函数求导 | $z_x=f_u u_x+f_v v_x$ | 有中间变量 |
| 隐函数求导 | $y'=-\dfrac{F_x}{F_y}$ | 函数藏在方程里 |
| 曲面切平面 | $z-z_0=f_x(x-x_0)+f_y(y-y_0)$ | 做几何应用 |
| 方向导数 | $D_{\mathbf e}f=\nabla f\cdot \mathbf e$ | 求指定方向变化率 |
| 梯度 | $\nabla f=(f_x,f_y)$ | 找最快上升方向、等值线法向 |
| 极值判别 | $D=f_{xx}f_{yy}-f_{xy}^2$ | 判断驻点类型 |
| 拉格朗日乘数 | $\nabla f=\lambda \nabla g$ | 做条件极值 |
| 二阶泰勒公式 | 常数项 + 一阶项 + 二阶项 + 高阶小量 | 局部近似、解释极值判别 |
| 最小二乘法 | 最小化 $\sum [y_i-(at_i+b)]^2$ | 由数据反推经验公式 |
