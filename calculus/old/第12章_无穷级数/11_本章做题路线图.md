# 第12章 无穷级数

## 本章做题路线图

### 1. 常数项级数怎么判

1. 先看一般项是否趋于 $0$。如果不趋于 $0$，立刻发散。
2. 如果是正项级数，优先在比较法、比值法、根值法、$p$ 级数之间选。
3. 如果是交错级数，先看能不能用莱布尼茨判别法。
4. 如果题目问“绝对收敛还是条件收敛”，就去看 $\sum |u_n|$。

### 2. 幂级数题怎么做

1. 先找展开中心。
2. 再求收敛半径。
3. 单独检查端点。
4. 如果要和函数，优先往已知展开式上靠，再考虑逐项积分或逐项求导。

### 3. 泰勒展开题怎么做

1. 能直接求导就直接法。
2. 导数太麻烦就间接法。
3. 写展开式时一定带成立区间。
4. 如果题目要求“证明展开成立”，记得看余项。

### 4. 傅里叶级数题怎么做

1. 先认周期。
2. 再看奇偶性。
3. 选完整区间公式或半区间公式。
4. 最后别忘记说明跳点处收敛到平均值。
