# 第11章 曲线积分与曲面积分

## 八、本章总结：遇到题该先想到什么

### 1. 公式选择表

| 题目特征 | 优先工具 |
| --- | --- |
| 曲线上积 $f\,\mathrm{d}s$ | 参数化，写出 $\mathrm{d}s$ |
| 有向曲线上积 $P\,\mathrm{d}x+Q\,\mathrm{d}y(+R\,\mathrm{d}z)$ | 参数化；若闭合且在平面上，优先 Green；若是空间闭曲线，优先 Stokes |
| 曲面上积 $f\,\mathrm{d}S$ | 选最方便的投影平面，化成二重积分 |
| 有向曲面上的通量 | 先看是不是闭曲面；闭曲面优先 Gauss |
| 判平面路径无关 | 看 $\dfrac{\partial P}{\partial y}=\dfrac{\partial Q}{\partial x}$，并检查单连通 |
| 判空间路径无关 | 看 $\nabla\times \mathbf{A}=0$，并检查一维单连通 |

### 2. 本章最容易混的四组概念

- 第一类曲线积分 vs 第二类曲线积分  
  前者积“标量沿长度”，后者积“向量沿位移”。

- 第一类曲面积分 vs 第二类曲面积分  
  前者积“标量沿面积”，后者积“法向通量”。

- 不看方向的积分 vs 看方向/侧的积分  
  $\mathrm{d}s,\mathrm{d}S$ 不带方向；$\mathrm{d}x,\mathrm{d}y,\mathrm{d}z,\mathrm{d}x\mathrm{d}y,\mathrm{d}y\mathrm{d}z,\mathrm{d}z\mathrm{d}x$ 带方向。

- Green / Gauss / Stokes  
  Green 是平面闭曲线，Gauss 是闭曲面，Stokes 是空间闭曲线和它张成的曲面。

### 3. 最后记住这 6 句话就够了

1. 第一类曲线积分本质是“函数值乘弧长”。
2. 第二类曲线积分本质是“场沿切向做功”，反向就变号。
3. 第一类曲面积分本质是“函数值乘面积”。
4. 第二类曲面积分本质是“法向通量”，换侧就变号。
5. Green 把平面闭曲线积分变成二重积分，Gauss 把闭曲面通量变成三重积分。
6. Stokes 把边界曲线的环流变成曲面上旋度的通量。
